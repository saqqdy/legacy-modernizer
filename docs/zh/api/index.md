# API 参考

Legacy Modernizer 编程式 API，可将迁移能力集成到你的工具链中。

## 安装

```bash
pnpm add legacy-modernizer
```

## 快速参考

| 函数 | 说明 |
|------|------|
| `scanFileContent()` | 扫描单个文件的遗留模式 |
| `scanProject()` | 扫描完整项目目录 |
| `renderAnalysisReport()` | 将分析报告渲染为 Markdown |
| `getDefaultConfig()` | 获取默认配置 |
| `mergeConfig()` | 合并用户配置与默认配置 |
