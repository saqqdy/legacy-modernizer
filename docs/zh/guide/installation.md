# 安装

## 方式 1: Claude Code Skill（推荐）

```bash
cd your-project
git clone https://github.com/saqqdy/legacy-modernizer.git .legacy-modernizer
cp -r .legacy-modernizer/.claude/skills/ .claude/skills/
```

## 方式 2: 编程式使用

```bash
pnpm add legacy-modernizer
```

## 前置条件

- **Node.js** >= 18
- **Claude Code** CLI（Skill 用法）
- **pnpm** 9+（开发用）
