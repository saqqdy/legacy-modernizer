# 交互式工作流

`/modernize` 命令提供引导式迁移体验：

## 1. 项目扫描

自动检测你的技术栈，扫描遗留模式。

## 2. 维度选择

选择要执行的迁移：

```
📦 可用的迁移维度：
[1] Vue 2 → Vue 3    (42 个模式, 18 个文件)
[2] JS → TypeScript  (35 个无类型文件)
[3] Webpack → Vite   (1 个配置文件)
[4] Jest → Vitest    (8 个测试文件)
[5] Vuex → Pinia     (2 个 store 文件)
```

## 3. 迁移计划

生成排序后的计划，包含风险评估。

## 4. 逐步执行

每个文件迁移展示：
- 完整 diff 预览
- 变更说明
- 确认提示（接受 / 跳过 / 编辑）

## 5. 完成汇总

迁移完成后：
- 统计信息（已迁移、已跳过的文件数）
- 建议后续操作（运行测试、构建检查）
- 迁移日志保存供参考
