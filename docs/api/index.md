# API Reference

Programmatic API for integrating Legacy Modernizer into your toolchain.

## Installation

```bash
pnpm add legacy-modernizer
```

## Quick Reference

| Function | Description |
|----------|-------------|
| `scanFileContent()` | Scan a single file for legacy patterns |
| `scanProject()` | Scan a full project directory |
| `renderAnalysisReport()` | Render analysis report to Markdown |
| `getDefaultConfig()` | Get default configuration |
| `mergeConfig()` | Merge user config with defaults |
